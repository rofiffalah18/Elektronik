Sensor

Sensor dalam konteks Arduino adalah perangkat fisik yang digunakan untuk mendeteksi atau mengukur berbagai parameter lingkungan, seperti suhu, cahaya, kelembaban, gerakan, dan banyak lagi. Sensor-sensor ini bekerja dengan cara mengubah kondisi fisik atau kimiawi menjadi sinyal listrik yang dapat dibaca oleh papan Arduino, sehingga memungkinkan kita untuk memonitor atau mengontrol berbagai aspek dalam proyek elektronika.

Cara Kerja

Sensor bekerja dengan mendeteksi perubahan dalam lingkungan sekitarnya, seperti suhu, cahaya, atau gerakan, dan mengonversikannya menjadi sinyal listrik. Papan Arduino membaca sinyal listrik yang dihasilkan oleh sensor dan menggunakan informasi tersebut untuk mengambil keputusan atau melakukan tindakan tertentu. Dengan menggunakan perangkat lunak (sketch) yang sesuai, pengguna dapat mengatur respons papan Arduino terhadap data yang diterima dari sensor.

Sensor Suhu

Mengukur tingkat kelembaban relatif dalam udara. Digunakan dalam proyek pemantauan kualitas udara dalam ruangan.

Sensor Kelembapan Tanah

Digunakan untuk mengukur tingkat kelembaban tanah dalam pot atau lahan pertanian. Berguna dalam proyek otomatisasi irigasi atau pemantauan kelembaban tanah.

Sensor Keasaman

Sensor Api

Sensor Getar
